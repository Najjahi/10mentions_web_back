BDD

Les bases de données sont les magasins qui pertmettent de stocker les informations et permettent notamment d'effectuer différentes opérations de lectures et d'écritures.

Elles sont efficaces pour retrouver des informations qui y sont stockées. Vous pouvez interroger une base de données, par exemple : combien d'utilisateurs sont inscrit sur mon site web ou quel article dans mon blog contient tel ou tel mot-clé.

La force des bases de données réside également dans la possibilité de lier les différentes informations qui y sont préalablement stockées.

SGBDD

SGBDD (Systéme de gestion de base de données) permet d'accuellir, d'exploiter et et de faire fonctionner les bases de données (moteur). Nous utilisons My SQL, d'autres SGBD existent : Oracle SQL serveur, PostgreSQL, ETC.

MySQL

MySQL (Systéme de gestion de base de données relationnel) C'est l'un des systémes les plus populaires au monde , il est très utilisé dans le monde du web.

Comme son nom l'indique, MySQL utilise SQL pour effectuer des requetes.

Toutes ces actions de gestion et manupilation passeront par une requête SQL.

SQL

SQL (Structured query langage) Langage de requête strucuré : est un langage informatique permettant d'intéragir avec des bases de données relationnelles..

Une fois que les informations on été enregistrées , il est important de pouvoir les gérer: ajout, modification, suppression, consultations.

PHPMyAdmin

PHPMyAdmin (souvent abrégé PMA) est une application de web de gestion pour MySQL. il est écrit en PHP et permet de gérer les BDD dans une interface visuelle simple et intuitive.

Il est important de noter que PHPMyAdmin n'est qu une aplication . Il est tout à fait possible de gérer MySQL avec d'utre applications similaires ou en ligne de commande.

CRUD

CRUD (Create, Read, Update, Delete), désigne les quatre opérations de base pour le stockage et la manupulation d'informations dans une base de données.

  • Create: insérer de nouveaux enregistrements (écritre);
  • Read: récupérer des enregistrements (lecture);
  • Apdate: Mettre à jour des enregistrements (écritre);
  • Delete: Supprimer des enregistrements (écritre);

Les opérations correspondent chacun à un type de requêtes SQL
